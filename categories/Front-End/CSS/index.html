flex 박스

flex 박스
  1. 자식 엘리먼트에는 어떤 것도 적지 말아야 한다. 반드시 부모 엘리먼트에만 적는다. 예를 들어, div태그를 컨트롤하기 위해서 div가 아닌 div를 감싸고 있는 body의 display 속성을 flex로 변경한다.
  2. 주축(main axis, 수평)은 justify-content로, 교차축(cross axis, 수직)은 align-items로 설정할 수 있다. 한편, flex-direction을 column으로 두는 경우 주축이 수직이되고 교차축이 수평이 된다.
  3. 만약 화면의 중심에 두고 싶은 경우에는 body의 height을 추가해준다. 이 때, height: 100vh 로 주면 전체화면의 중앙에 놓이게 된다. (vh = viewport height = screen height)

class에 관하여

class에 관하여

class는 여러 동일한 태그를 사용할 때 일부에만 원하는 스타일을 적용하고 싶은 경우에 사용한다. 기존 태그에 원하는 스타일을 적용하려고 하다보면 id를 이용해서 설정할 수 있으나, 이런 경우에는 매번 새로운 id를 만들고 이에 대해 스타일을 적용해 주어야 하는 번거로움이 생긴다. class를 사용하면 이러한 번거로움 없이 간단하게 스타일을 적용할 수 있다.

class는 “.class명” 명으로 입력한다. 그리고 원하는 태그 안에 class=”class명” 으로 적용하면 된다. 또한, class는 여러개를 가질 수 있다. class=”class1 class2 class3” 으로 적용이 가능하다.

박스태그

box 태그에 관하여

box 태그는 페이지 UI를 구성하는데 있어서 매우 기본적이고 중요한 태그이다. box 태그 특징은 옆에 다른 요소가 올 수 없다는 점이다. 우리는 이를 block이라고 부른다. box 태그의 종류에는 div, p, main, header, footer 등이 존재한다. 반면, 옆에 다른 요소가 올 수 있는 태그들이 존재한다. 우리는 이를 inline이라고 부르며 대표적인 것으로 span, a 등이 존재한다. block은 높이와 너비가 존재하지만, inline은 높이와 너비가 없다.

한편, block, inline 특성을 가진 태그에 대해 display 속성을 사용해서 바꿀 수 있다. 예를 들어, div 태그에 display 속성으로 inline을 주면 옆에 다른 요소가 올 수 있게 된다.

box의 특징은 크게 3가지가 있다. margin, padding, border이다.

margin

box의 border(경계) 바깥에 있는 공간을 의미한다.

margin: 20px // 상하좌우 20px

margin: 20px 15px // 상하 20px 좌우 15px

margin 20px 5px 12px 9px // 상 20px 우 5px 하 12px 좌 9px(시계 방향)

collapsing margins? 상하에 대해서 box간 경계가 일치하는 경우에 margin이 같이 적용되는 현상

padding

box의 border(경계) 안쪽에 있는 공간을 의미한다.

CSS(Cascading Style Sheet) 작성법

CSS(Cascading Style Sheet) 작성하는 방법은 다음과 같다.


  1. selector 이름(태그명 또는 #아이디) 적고 중괄호 열기
  2. 중괄호 안에 원하는 속성 입력
  3. html에서 불러오기


여담으로 Cascading을 번역하면 “계단식”인데, 이는 style이 코드의 위에서부터 아래로 순서대로 적용된다는 뜻이라고 한다. 즉, 겹치는 속성이 있으면 가장 마지막에 작성된 style이 적용된다는 말!
